# ربات تلگرام دخل‌وخرج خانواده

ربات تلگرامی برای ثبت درآمد/هزینه روزانه، نمایش باقیمانده بودجه، دسته‌بندی خودکار خریدها، گزارش هفتگی/ماهانه، و لیست خرید هوشمند که با عکس فاکتور تطبیق داده می‌شود. چندنفره است — هر عضو خانواده با یک کد دعوت به همان حساب مشترک وصل می‌شود و از گوشی خودش کار می‌کند.

## منو

با زدن `/start` یا `/menu` یک کیبورد دکمه‌ای پایین صفحه باز می‌شود (💰 موجودی، ➖ ثبت هزینه، ➕ ثبت درآمد، 🛒 لیست خرید، 📊 گزارش، ⚙️ تنظیمات، ❓ راهنما) — نیازی به حفظ‌کردن دستورها نیست، کاربر فقط دکمه می‌زند. دکمه‌های ⚙️ تنظیمات و 📊 گزارش هم خودشان زیرمنوی دکمه‌ای (inline) باز می‌کنند. علاوه بر این، آیکون منوی داخلی تلگرام (کنار جعبه پیام) هم لیست کامل دستورها را نشان می‌دهد.

## دستورات ربات

| دستور | کاربرد |
|---|---|
| `/start` | ساخت حساب خانواده (بار اول) |
| `/invite` | گرفتن کد دعوت برای بقیه اعضا |
| `/join CODE` | پیوستن به خانواده‌ای دیگر با کد دعوت |
| `/budget 5000000` | تنظیم بودجه بازه جاری |
| `/budget +500000` یا `/budget -200000` | افزایش/کاهش بودجه فعلی |
| `/income 2000000 حقوق` | ثبت درآمد |
| `/expense 50000 نان` | ثبت هزینه (یا فقط بنویس: `نان 50000`) |
| `/balance` | باقیمانده بودجه، هزینه امروز/هفته/ماه |
| `/report month` | گزارش دسته‌بندی‌شده (`day`/`week`/`month`) |
| `/transactions` | نمایش تراکنش‌های اخیر با دکمه حذف/ویرایش مبلغ |
| `/currency EUR` یا `/currency یورو` | تغییر واحد پول (پیش‌فرض: تومان — هر خانواده جدا قابل تنظیم است) |
| `/period monthly` یا `/period weekly <روز>` | انتخاب بازه بودجه: ماهانه یا هفتگی (با روز شروع دلخواه) |
| `/newlist` سپس هر آیتم یک خط، بعد `/donelist` | ساخت لیست خرید |
| `/list` | نمایش لیست با دکمه‌های تیک‌زدن دستی |
| عکس فاکتور یا فایل PDF فاکتور | استخراج خودکار قلم‌ها و قیمت‌ها، ثبت هزینه، و تطبیق با لیست خرید فعال |
